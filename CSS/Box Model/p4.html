CSS Box Model...!

Every element that can be displayed on a web page is comprised of one or more rectangular boxes. Css Box model typically describes how these rectangular boxes are laid out on a web page. These boxes can have diff Properties and can interact with each other in different ways, but every box has content area & Optional surroundings padding border and margin areas
