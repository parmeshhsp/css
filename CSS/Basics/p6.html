Defining Colors

Colors in CSS most often Specifies in following formats...

  1. A color Keywords like - "red","yellow","Blue" etc
  2. A HexValue like-"#ff00","#00ff0"
  3. An RGB Value (Red Green Blue) like - RGB (255,0,0)
